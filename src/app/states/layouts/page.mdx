# layouts

An [app state](/guides/flux-architecture) of the layouts.
You can obtain the `layouts` state and integrate it with React components.{{ className: 'lead' }}

You can change the layout components with [Layout Manager](/modules/layout-manager).

---

## Data Structure
<Row>
  <Col>
    A map of `Record<string, LayoutComponents>` where:

    - `string`: The layout name
    - `LayoutComponents`: An array of React component class names
  </Col>
  <Col sticky>
    Get the `layouts` state manually:

    ```js
    const { layouts } = inkdrop.store.getState()
    ```

    Connect the state with your React component:

    ```js
    import { useSelector } from 'react-redux'
    const selector = ({ layouts }) => layouts
    const MyComponent = props => {
      const layouts = useSelector(selector)
      // render
    }
    ```

    ```js {{ title: 'Example values' }}
    {
      "main:full": [
        "ModalLayout",
        "SideBarLayout",
        "NoteListBarLayout",
        "EditorLayout"
      ],
      "main:slim": [
        "ModalLayout",
        "NoteListBarLayout",
        "EditorLayout"
      ],
      "main:distraction-free": [
        "ModalLayout",
        "EditorLayout"
      ],
      "sidebar": [
        ...
    }
    ```
  </Col>
</Row>
